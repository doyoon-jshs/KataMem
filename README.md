# KanaMem

히라가나 46자를 짧은 카드 학습과 빠른 퀴즈로 외우는 심플한 정적 웹앱입니다.

## 실행

브라우저에서 `index.html`을 열면 바로 사용할 수 있습니다.

## 무료 배포

### GitHub Pages

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 올립니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Deploy from a branch`를 선택하고 `main` 브랜치의 `/root`를 지정합니다.
5. 저장 후 표시되는 Pages 주소로 접속합니다.

### Netlify

1. [Netlify Drop](https://app.netlify.com/drop)에 접속합니다.
2. 이 폴더 전체를 드래그해서 올립니다.
3. 생성된 주소로 바로 접속합니다.

빌드 과정이 없는 정적 사이트라 무료 호스팅에 잘 맞습니다.
