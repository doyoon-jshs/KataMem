# KanaMem

히라가나를 표와 퀴즈로 가볍게 외우는 웹앱입니다.

## 링크

[https://doyoon-jshs.github.io/KanaMem/](https://doyoon-jshs.github.io/KanaMem/)

## 기능

- 히라가나 46자 표
- 아는 글자 표시
- 카드형 퀴즈
- 전체표 퀴즈
- 브라우저에 진척도 저장

## 사용

1. 표에서 이미 아는 글자를 눌러 표시합니다.
2. 퀴즈에서 발음과 글자를 맞힙니다.
3. 틀린 글자는 표에 복습 필요로 표시됩니다.

진척도는 사용하는 브라우저에만 저장됩니다.
